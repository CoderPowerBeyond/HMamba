# HMamba
HMamba: Hyperbolic Mamba for Sequential Recommendation

## Usage

### Requirements

* Python: 3.8.10
* PyTorch: 2.0.0+cu118
* CUDA 11.8
* Install RecBole:
  * `pip install recbole`
* Install causal Conv1d and the core Mamba package:
  * `pip install causal-conv1d>=1.2.0`
  * `pip install mamba-ssm`

### Run

```python run.py```
